# 🛡️ UPI Fraud Analysis — End-to-End Data Analytics Project

> Detect • Analyze • Visualize  
> A complete end-to-end data analytics project focused on identifying fraudulent UPI transactions using Python, Excel, and Power BI.

---

# 📌 Project Overview

This project demonstrates a complete data analytics workflow for UPI (Unified Payments Interface) fraud analysis in India. The solution includes synthetic transaction generation, data cleaning, feature engineering, exploratory data analysis (EDA), and interactive dashboard creation using Power BI.

The project is designed with an industry-oriented approach to simulate real-world digital payment fraud analysis scenarios.

## Key Dataset Metrics

| Metric | Value |
|---|---|
| Total Transactions | 5,000 |
| Fraud Transactions | ~384 |
| Fraud Rate | 7.68% |
| Total Fraud Amount | ₹4.49 Crore |
| Average Fraud Amount | ₹1.17 Lakh |
| Peak Fraud Hour | 10 PM |
| Data Period | Jan 2024 – Dec 2024 |
